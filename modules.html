Project Modules

This project has declared the following modules:

Name Description
Eclipse Sisu Container JSR330-based container; supports classpath scanning, auto-binding, and dynamic auto-wiring
Eclipse Sisu Plexus Plexus-JSR330 adapter; adds Plexus support to the Sisu-Inject container
Eclipse Sisu Maven Plugin Maven plugin generating Sisu index files
Eclipse Sisu JSR-330 Extender Automatically discovers and wires JSR-330 annotated Sisu components contained in OSGi bundles
org.eclipse.sisu.plexus.extender Plexus-JSR330 adapter; adds Plexus support to the Sisu-Inject container
