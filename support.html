Mailing Lists

If you have a question about using Sisu that the existing documentation nor the forum does not answer, we'd like to hear from you on the user mailing list or developer mailing list. The latter should be used for questions about the internal design of Sisu.

Before posting your question, please search the mailing list archives in case your question has been brought up before and was already answered.

Forum

There is a dedicated Sisu discussion forum @ Eclipse Community Forums where questions can be asked as alternative to using the mailing lists.

Issue Tracker

If you encounter a bug or have a feature request, please open a sufficiently detailed issue over in the project's issue management. We especially appreciate reports that come with suggested patches or enough information to be recreated locally. Consider also the information about Contributing
